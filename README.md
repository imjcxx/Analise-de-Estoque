# 📦 Analise-de-Estoque

Sistema desenvolvido em Java focado no controle de qualidade e identificação de itens defeituosos em inventários de estoque. Projeto prático desenvolvido durante o primeiro semestre da faculdade.

## 🎯 Funcionalidades
- Leitura automatizada da quantidade de produtos por lote.
- Somatória total de itens registrados.
- Critério de parada dinâmico utilizando loops estruturados.

## 🛠️ Tecnologias e Conceitos Aplicados
- **Linguagem:** Java
- **Conceitos:** Estrutura de repetição (`while`), entrada de dados com `Scanner` e lógica de contagem/acumulação de variáveis.

## 💻 Como Executar o Projeto
1. Certifique-se de ter o JDK (Java Development Kit) instalado.
2. Baixe o arquivo `analiseEstoque.java`.
3. Abra e execute o arquivo na sua IDE de preferência (como o Apache NetBeans) ou compile direto pelo terminal.
